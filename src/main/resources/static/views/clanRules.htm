Clan rules

  1. Προσέχουμε τι λεμε στα βιντεακια που κανουμε share! Δεν κανουμε φυλετικες διακρίσεις. Βρισιες επιτρέπονται μεχρι ενος ορίου

  2. Δεν ρωτάμε συνέχεια να γινουμε elder και co, θα το κανουμε οι παλαιότεροι οταν πρέπει.

  3. Oποιος βρίσει μέλος τρώει μπαν εκτός αν εχει σοβαρο λόγο ο οποίος θα πρέπει να διευκρινιστεί!

  4. Εχουμε τουλαχιστον 10cc και 200 donates κατα μεσον ορο.
